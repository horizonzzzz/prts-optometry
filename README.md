# PRTS Optometry

An unofficial Arknights fan page created for learning and fan creation.

## Development

```bash
npm install
npm run dev
npm test
npm run typecheck
npm run build
```

Use `npm run preview` for a local Worker preview and `npm run deploy` to deploy.
